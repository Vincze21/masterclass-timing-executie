# Masterclass: Timing și Execuție în Investiții

Prezentare interactivă pentru masterclass despre timing educațional și execuție sistematică în investiții.

## Conținut

- **Hook**: Comparație portofolii identice cu rezultate diferite (5.000€ → rezultate diferite prin timing)
- **DCA Adaptat**: Strategii de Dollar Cost Averaging optimizat vs clasic
- **Momentum**: Analiza momentum folosind ROC și SMA200
- **Sezonalitate**: Pattern-uri istorice S&P 500 (1950-2025)
- **Calendar 2026**: Ferestre favorabile și evenimente majore (FOMC, NFP, CPI)
- **Rebalansare Tactică**: Control risc prin rebalansare (nu maximizare randament)
- **Checklist Execuție**: Sistem de verificare înainte de intrare

## Utilizare

1. Descarcă sau clonează repository-ul
2. Deschide `index.html` în browser
3. Scroll prin prezentare pentru a vizualiza toate conceptele
